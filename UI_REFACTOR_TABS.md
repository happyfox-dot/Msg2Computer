# 桌面端 UI 重构 - 标签页布局

## 📅 更新时间
2026/06/09

## ✨ 新功能

### 标签页导航
将原有的单一面板拆分为 5 个独立标签页，提升信息组织的清晰度：

1. **📋 短信验证码** - 默认首页，显示所有短信验证码
2. **⏱️ 动态验证码** - 显示所有 TOTP 验证码
3. **📲 已授权手机** - 管理已配对的手机设备（带实时数量角标）
4. **🔗 配对** - 二维码配对页面
5. **⚙️ 设置** - 连接状态、统计信息、版本号、项目链接

### 改进点

#### 视觉优化
- 标签页采用图标 + 文字布局，直观清晰
- 已授权手机标签页显示数量角标（红色圆形徽章）
- 每个标签页独立滚动区域，避免内容溢出
- 配对页面居中显示，布局更合理

#### 功能增强
- **统计信息**：设置页面显示已授权手机数、短信验证码数、TOTP 验证码数
- **连接状态**：状态栏和设置页面同时显示连接状态
- **外部链接**：设置页面可以直接打开 GitHub 项目地址

#### 交互改进
- 标签页切换流畅，保留当前状态
- 手机数量角标实时更新
- 统计信息随数据变化自动刷新

## 📊 用户体验提升

### 信息架构更清晰
- **之前**：所有功能挤在一个页面，需要上下滚动查找
- **现在**：按功能分类到独立标签页，一键切换

### 多设备场景友好
- **多手机**：独立的"已授权手机"标签页，数量角标实时显示
- **多验证码**：短信和 TOTP 分开显示，不再混杂
- **多电脑**：配对页面独立，不影响其他功能

### 统计信息可视化
- 实时显示各类数据数量
- 连接状态在状态栏和设置页面双重显示
- 便于诊断和了解使用情况

## 🔧 技术实现

### 标签页切换
- CSS 绝对定位 + `display: none/flex` 控制面板显示
- 点击标签按钮时切换 `.active` 类
- 保留各面板状态，切换时不重新渲染

### 数量角标
- 实时监听 `authorizedPhones` 数组变化
- 动态更新角标显示和隐藏
- 红色圆形徽章，视觉突出

### 统计信息更新
- 在数据变化时调用 `updateSettingsStats()`
- 包括：新验证码、手机状态变化、TOTP 更新
- 自动同步到设置页面

## ⚠️ 注意事项

1. **兼容性**：保持所有原有功能不变，只是重新组织布局
2. **性能**：标签页切换使用 CSS，不触发重新渲染
3. **状态保留**：切换标签页时保留各页面状态（滚动位置、数据等）
